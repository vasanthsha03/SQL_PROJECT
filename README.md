# 👨‍💼 Employee Management System – SQL Case Study

## 📌 Project Overview

This project demonstrates a complete Employee Management System built using MySQL.
It covers database design, data insertion, and 30 structured SQL queries ranging from Basic to Advanced level.

## The objective of this case study is to strengthen SQL concepts including:

CRUD Operations

Joins

Aggregate Functions

Subqueries

Group By & Having

Window Functions

Real-world Business Scenarios

🗂️ Database Schema
1️⃣ Department Table

dept_id (Primary Key)

dept_name

2️⃣ Employee Table

emp_id (Primary Key)

emp_name

salary

hire_date

dept_id (Foreign Key)

Relationship:
One Department → Many Employees
